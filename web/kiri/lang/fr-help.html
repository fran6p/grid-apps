Kiri:Moto est un logiciel de découpe unique, fonctionnant dans un navigateur, entièrement open source, qui transforme les modèles 3d en instructions de machine pour des imprimantes 3D, des CNC, et des graveurs laser.


Le fait de fonctionner dans le navigateur signifie que tout le processus de découpe a lieu en local, et non dans un nuage. Vos données et modèles restent totalement privés.


Kiri se comporte comme une application native de votre ordinateur avec en plus la sécurité de l'exécution dans le bac à sable du navigateur.


L'autre avantage de fonctionner à partir du web est que Kiri est toujours à jour. Les nouvelles fonctionnalités, corrections de bogues et mises à jour sont fréquemment mises en place. Si vous souhaitez exécuter une version de Kiri ancienne ou une de développement, sélectionnez celle-ci dans le menu utilisateur en haut à droite.

Kiri vient de Kiri-e qui est l'art Japonais du découpage de papiers Moto est un raccourci pour "Modeling Tool" (Outil de modélisation)
Guide de démarrage rapide

  • voir The Definitive Video Guide to Kiri:Moto 2.5
  • le menu setup sélectionne le type de matériel et un appareil
  • les préférences permettent de régler l'interface et les caractéristiques de l'appareil
  • personnaliser un appareil si vous souhaitez modifier les paramètres par défaut
  • ouvre un fichier ou importe des pièces dans l'espace de travail
  • utiliser les menus de droite pour personnaliser les paramètres de tranchage
  • trancher les objets pour voir l'effet des paramètres
  • la prévisualisation calcule les données de cheminement des objets tranchés
  • l'exportation convertit les chemins d'accès aux fichiers GCode et SVG

    • Modes de fonctionnement

    Chaque mode (FDM, CAM, Laser, SLA) a des caractéristiques et des paramètres différents. Explorez les vidéos Youtube pour en savoir plus.

    Options complémentaires

    Dans le menu utilisateur en haut à droite, utiliser l'option export pour créer une copie téléchargeable des profils du matériel et (optionnellement) de l'espace de travail. Ce fichier pourra ensuite être utilisé pour contrôler et rétablir un instantané de votre installation à tout moment. Il peut également être partagé afin que les développeurs et autres utilisateurs puissent reproduire votre environnement pour aider au débogage.

    Open Source

    Kiri:Moto est totalement open-source. Consultez la page du projet Github. Il existe des projets complémentaires, tant sur le plan matériel que logiciel. Les nouveaux développeurs et les contributions au code sont les bienvenus à tout moment. La licence du code est celle du MIT.


    Toute donation est également bienvenue.


    Intégrations

    Kiri:Moto est également intégré dans plusieurs autres applications web

  • Onshape via son magasin d'Applications
  • Thingiverse en tant que native Thing Apps
  • SimplyPrint.io comme trancheur principal (en cours)
  • CAMLab est une branche d'une version antérieure de Kiri:Moto
