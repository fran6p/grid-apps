Kiri:Moto is a unique browser-based, fully open source slicer that turns 3d models into machine instructions for 3D printers, CNC mills, and Laser Cutters.


Being browser-based means that all processing happens in your local browser, and not in the cloud. Your data and models stay private to you.


This means that Kiri behaves just like a native app on your computer with the added safety of running inside the browser's security sandbox.


The other benefit of running from the web is that Kiri is always up to date. New features, bug fixes, and updaets roll out frequently. If you want to run an older or development version of Kiri, select that version from the user menu on the top right.

Kiri comes from Kiri-e which is the Japanese art of papercutting Moto is short for "Modeling Tool"
Quick Start Guide

  • watch The Definitive Video Guide to Kiri:Moto 2.5
  • setup menu selects a device and mode
  • preferences allow interface and mode-specific settings
  • customize a device if you want to change default settings
  • open files or import parts to the workspace
  • use the right-side profile menus to customize slicing parameters
  • slice objects to see the effect of the parameters
  • preview calculates path data from sliced objects
  • export converts paths to GCode and SVG files

    • Operating Modes

    Each mode (FDM, CAM, Laser, SLA) has different features and settings. Explore the Youtube videos to find out more.

    In-App Options

    Under the user menu on the top right, use the export option to create a downloadable copy of your device profiles and (optionally) workspace. This file can be used to checkpoint and restore a full snapshot of your setup an any time. It can also be shared so that developers and other users can replicate your environment to help aid in debugging.

    Open Source

    Kiri:Moto is entirely open-source. Check out the Github project page. There are companion projects, both hardware and software. New developers and code contributions are welcome at any time. The code license is MIT.


    Donations are also welcome.


    Integrations

    Kiri:Moto is also integrated into several other web-based applications

  • Onshape in the App store
  • Thingiverse as a native Thing Apps
  • SimplyPrint.io as the primary slicer under the covers
  • CAMLab is a fork of an earlier version of Kiri:Moto
