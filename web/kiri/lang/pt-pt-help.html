Kiri:Moto é um slicer browser-based único e de código aberto que transforma modelos 3d em instruções-máquina para impressoras 3D, fresadoras CNC, e Cortadoras Laser.


Ser browser-based significa que todo o processamento acontece no seu browser de modo local, e não na cloud. Seus dados e modelos se mantem privados para você.


Isto significa que a Kiri se comporta mesmo como uma app nativa no seu computador com a segurança adicional de correr dentro do sandbox de segurança do seu browser.


O outro benefício de correr a partir da internet é que a Kiri está sempre actualizada. Novas funcionalidades, correções de bugs e updates são disponibilizadas frequentemente. Se você quiser usar uma versão mais antiga ou em desenvolvimento da Kiri, selecione essa versão a partir do menu de usuário no topo à direita.

    Opções dentro da App

    Por baixo do do menu do usuário no topo à direita, use a export opção de criar uma copia do seu dispositivo que poderá ser baixada perfis e (opcional) o espaço de trabalho. Este Ficheiropode ser usado para restaurar uma fotografia completa das suas definições e configurações a qualquer momento. Também poderá ser partilhado para que os desenvolvedores e outros usuários possam replicar o seu ambiente para ajudar a debugar.

    Fonte Aberta

    Kiri:Moto é completamente fonte aberta. Dê uma olhada na página do projeto no GitHub. Existem projetos irmãos, tanto de hardware como de software. Novos desenvolvedores e contribuições são bem-vindos a qualquer momento. A licenla de código é MIT.


    Donativos são igualmente bem-vindos.

  • Thingiverse como uma Thing App nativa
